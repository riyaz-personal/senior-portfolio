# Senior Portfolio (React + Tailwind)

A premium, minimal portfolio for senior frontend engineers.

## Tech
- React + Vite
- Tailwind CSS
- Framer Motion

## Quickstart
```bash
bun i # or npm i / pnpm i / yarn
bun run dev # or npm run dev / pnpm dev / yarn dev
```

## Build & Preview
```bash
bun run build && bun run preview
```

## Customize
- Update **src/data.js** with your real experience, projects, skills, testimonials.
- Replace social links in **Hero** and **Contact**.
- Put your **resume.pdf** in `/public`.

## Deploy
- Push to GitHub and deploy on Vercel or Netlify.
